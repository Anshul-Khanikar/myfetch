# myfetch 🟢

A lightweight, seamless wrapper that combines the deep system diagnostics of **Fastfetch** with the spinning 3D ASCII art of **areofyl/fetch**. 

Instead of choosing between rich information and a cool visual, `myfetch` seamlessly renders both in a single terminal command, updating dynamic stats (like RAM and Uptime) in real-time.

## 🚀 Installation

You can install `myfetch` on any Debian/Ubuntu-based system in just a few steps:

1. Clone this repository:

   git clone [https://github.com/Anshul-Khanikar/myfetch.git](https://github.com/Anshul-Khanikar/myfetch.git)

2. Navigate into the directory:

    cd myfetch

3. Run the installer:

    ./install.sh

🛠️ Usage

Simply type myfetch in your terminal!

Press any key to exit the 3D animation. The temporary visual buffer will cleanly close, leaving a standard Fastfetch output seamlessly in your terminal history.
⚙️ Configuration

myfetch uses its own dedicated configuration folders so it won't conflict with standard Fastfetch setups.

    Change the Logo: Edit ~/.config/myfetch/logo.txt

    Change the Modules: Edit ~/.config/myfetch/config
